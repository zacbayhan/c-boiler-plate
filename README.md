#  Boiler Plate...

A quick and easy way to start a c program, complete with make file. Put .c, .h, .cpp files into the src directory


* Getting Started

```
* git clone git@github.com:zacbayhan/c-boiler-plate.git

* cd c-boiler-plate

* make

* ./build/a.out

* make clean

```
